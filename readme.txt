Statistico – standalone stats and analytics for WordPress

== Description ==

Statistico is unique WordPress plugin of its kind which helps to track and analyze most essential, useful and highly interesting visitor's data.

It focuses on collecting cruical website visitors data, storing and representing it in the right way so you can make quick and precise decisions.

**STATISTICO FEATURES:**

* Browser agent and browser version
* Platform / Operating system
* Visitor’s Screen resolution
* Browser usage
* OS Usage
* Fancy Column and Pie charts
* One-click setup
* Easy integration

Visit [WP Salad](http://wpsalad.com/)

== Installation ==

1. Upload `statistico.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
